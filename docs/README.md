# SmartMoney Documentation

This folder is the main knowledge base for the SmartMoney project.

## Files

- `PRODUCT.md`  
  Product purpose, MVP scope, users, and business flow.

- `ARCHITECTURE.md`  
  Technology stack, project structure, dependency rules, and major decisions.

- `ENGINEERING_RULES.md`  
  Coding standards, security rules, Git workflow, and Definition of Done.

- `ROADMAP.md`  
  Milestones, sprint plan, backlog, and current status.

- `WORK_LOG.md`  
  Simple history of completed work, decisions, commits, and next tasks.

## Maintenance Rule

Update only the file affected by the current work.

Do not duplicate the same information in multiple files.
